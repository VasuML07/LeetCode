## 📖 About This Repository

A curated collection of **150 carefully selected LeetCode problems** organized by **10 essential DSA patterns**. Each pattern contains 15 questions with a structured progression from Easy to Hard, designed specifically for FAANG-level interview preparation.

**Why Pattern-Based Learning?**
- Recognize problem types instantly during interviews
- Build muscle memory for common solution approaches
- Efficient preparation vs. random problem solving

## 🎯 Pattern Overview

| # | Pattern | Description |
|---|---------|-------------|
| 01 | **Sliding Window** | Optimal for substring/subarray problems with contiguous elements |
| 02 | **Two Pointers** | Efficient traversal from both ends or finding pairs |
| 03 | **Binary Search** | O(log n) search in sorted arrays or answer spaces |
| 04 | **DFS / BFS** | Graph/tree traversal and connectivity problems |
| 05 | **Dynamic Programming** | Optimization through overlapping subproblems |
| 06 | **Backtracking** | Exhaustive search with pruning for constraint problems |
| 07 | **Greedy** | Locally optimal choices for global optimization |
| 08 | **Heap / Priority Queue** | Efficient min/max operations and top-k problems |
| 09 | **Graphs** | Advanced graph algorithms (Dijkstra, Union-Find, etc.) |
| 10 | **Stack / Monotonic Stack** | LIFO structure for next greater/smaller element problems |

## 🚀 How to Use This Repository

1. **Start with one pattern** - Master the core concept before moving on
2. **Follow the order** - Questions are arranged Easy → Medium → Hard
3. **Track progress** - Update status from ⬜ to ✅ after solving
4. **Time yourself** - Aim for 20-30 mins (Easy), 30-45 mins (Medium), 45-60 mins (Hard)
5. **Review solutions** - Understand multiple approaches for each problem
